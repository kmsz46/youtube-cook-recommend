# SUSURU巡り
SUSURU TV(https://www.youtube.com/@SUSURUTV)を対象とした検索支援アプリの開発
SUSURU TVで詳細されている店舗をGoogle Map上にプロットし、選択することでYouTubeの動画を視聴できるアプリです。

## 使用技術
### データの整形
- Python
- YouTube Data API
- Geocording API
### フロントエンド
- React+TypeScript
- Vite
- Material UI(CSSフレームワーク)
- Google Map API
### デプロイ
- Firebase Hosting
- Firestore
- Cloud Function
